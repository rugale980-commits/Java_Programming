# Assignment 34 :

This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Write program Which display ASCII table. Deci,Hexa,Octal.
2. Accept character from user. IF character is small display its corresponding capital character and in other case display as its.
3. Accept character from user. IF character is small display all character from the input character till Z. in same small reverse order.
4. Accept character from user and check whether it is special or not.
5. Accept character from user and display its ASCII value in decimal octal and hexadecimal format.

Commands used to execute these codes are:-

javac FileName.java

java FileName
