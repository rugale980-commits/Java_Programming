# Assignment 33 :
This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept Character from user and check whether it is aplhabet or not (A-Z a-z)
2. Accept Character from user and check whether it is Capita or not (A-Z)
3. Accept Character from user and check whether it is Digit or not (0 - 9)
4. Accept Character from user and check whether it is Small case or not (a - z)
5. Accept division of student from user & depends on the division display Exam timing.
 
Commands used to execute these codes are:-

javac FileName.java

java FileName
