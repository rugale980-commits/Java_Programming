# Assignment 35 :

This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept String from user and count number of capital character.
2. Accept String from user and count number of Small character.
3. Accept String from user and return Small & capital diffrence.
4. Accept String from user & check whether it cintain vowel or not.
5. Accept String from user & Display it in reverse order.

Commands used to execute these codes are:-
javac FileName.java

java FileName
