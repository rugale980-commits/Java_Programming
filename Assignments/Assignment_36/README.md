# Assignment 36 :

This folder contains problems related to digits in which input is taken from user as a Integer, problems are:-

1. Accept String from user & convert it into lower case.
2. Accept String from user & convert it into upper case.
3. Accept String from user & toggle case.
4. Accept String from user & display only digit from that string.
5. Accept String from user & Count number of white spaces.

Commands used to execute these codes are:-
javac FileName.java

java FileName
